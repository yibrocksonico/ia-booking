1. Tipografías Seleccionadas (Google Fonts)
Buscamos un equilibrio perfecto entre legibilidad extrema y estética de ciencia ficción:
Tipografía de Título / Display: Space Grotesk
Estilo: Sans-serif geométrica con un toque técnico y futurista.
Uso: Títulos principales, llamadas a la acción (CTAs) y cabeceras de gran tamaño.
Enlace para descargar: Space Grotesk en Google Fonts
Tipografía de Cuerpo / Botones: Inter
Estilo: Una de las tipografías modernas más limpias del mundo. Ofrece una nitidez impecable para textos descriptivos pequeños o medianos.
Uso: Párrafos de descripción, menús, etiquetas y formularios de reserva.
Enlace para descargar: Inter en Google Fonts
Tipografía de Datos / Métricas: JetBrains Mono
Estilo: Tipografía de ancho fijo (monoespaciada) con un aire cyberpunk o de "consola de comando".
Uso: Códigos de reserva, relojes de tiempo real, temperatura de cabinas, números de pod y especificaciones de hardware.
Enlace para descargar: JetBrains Mono en Google Fonts

2. Paleta de Colores (Códigos HEX para Creadores Visuales)
Diseñamos un esquema de color de alto contraste sobre un lienzo oscuro para simular una interfaz espacial, complementada con acentos neón sumamente llamativos:
Color	Código HEX	Descripción / Uso sugerido en tus Flyers
Cian Neón	#00eaff	Color estrella de la marca. Úsalo para bordes resplandecientes, títulos clave y botones destacados.
Azul Eléctrico	#0055ff	Combínalo en gradientes con el Cian Neón para dar volumen holográfico a tus vectores.
Índigo Profundo	#070b19	Excelente para el fondo de paneles interiores, cajas de ventajas o fondos de folletos.
Pizarra Oscuro	#151a2e	Para cards secundarias o marcos del folleto.
Negro Medianoche	#020409	El color de fondo dominante. Genera el contraste perfecto necesario para que las luces cian brillen físicamente en papel satinado.
Colores de Acento Secundarios (Para las secciones del Flyer):
Oro / Ámbar (#f59e0b): Representa el área acogedora y Premium del Coworking (café, calidez, concentración).
Magenta Neón (#ff0096): Ofrece un contraste vanguardista excelente para indicar alertas de seguridad, luces de personalización o zonas de relax.
